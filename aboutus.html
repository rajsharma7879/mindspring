About MindSpring

A College Project for Web Development

Project Overview

MindSpring was developed as a mini project for college. Our goal was to create a dynamic, interactive, and user-friendly quiz platform. It leverages modern web technologies to provide a seamless learning experience, complete with user authentication, progress tracking through a badge system, and a vast, expandable question bank, all powered by a Firebase backend.
